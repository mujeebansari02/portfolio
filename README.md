# Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and experience.

## Features

- Modern dark theme design
- Responsive layout for all devices
- Interactive sections:
  - Hero/About section
  - Experience
  - Projects
  - Certificates
  - Education
  - Contact
- Smooth animations and transitions
- Mobile-friendly navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

## Sections

1. **Hero Section**
   - Profile image
   - Personal introduction
   - Contact information

2. **Experience Section**
   - Professional experience
   - Skills and achievements

3. **Projects Section**
   - Project showcase
   - Technology stack
   - Live demos and source code

4. **Certificates Section**
   - Professional certifications
   - Course completions

5. **Education Section**
   - Academic background
   - Institution logos

6. **Contact Section**
   - Contact form
   - Social media links

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/mujeebansari02/portfolio.git
   ```

2. Open `index.html` in your web browser
