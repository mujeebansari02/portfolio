# Modern Portfolio Website

A fully responsive, visually stunning portfolio website built with HTML, CSS, and JavaScript.

## Features

- Modern and clean design
- Fully responsive layout
- Smooth scrolling navigation
- Interactive skill bars
- Form validation
- Fade-in animations
- Mobile-friendly navigation

## Setup Instructions

1. Clone this repository to your local machine
2. Add the following images to the root directory:
   - `hero-bg.jpg` - A high-quality background image for the hero section (recommended size: 1920x1080px)
   - `placeholder-profile.jpg` - Your profile picture (recommended size: 400x400px)
   - `spotify-clone.jpg` - Project thumbnail for Spotify Clone (recommended size: 800x600px)

## Image Requirements

### Hero Background (hero-bg.jpg)
- Minimum resolution: 1920x1080px
- Format: JPG
- Content: Choose a professional, minimalist background that represents your work style

### Profile Picture (placeholder-profile.jpg)
- Recommended size: 400x400px
- Format: JPG
- Content: A professional headshot with a clean background

### Project Thumbnail (spotify-clone.jpg)
- Recommended size: 800x600px
- Format: JPG
- Content: Screenshot or representation of your Spotify Clone project

## Customization

1. Colors: Edit the CSS variables in the `:root` selector in `styles.css`
2. Content: Update the text content in `index.html`
3. Links: Replace all social media and project links with your own
4. Images: Replace the placeholder images with your own images

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Contact Form Setup

The contact form currently shows a success message but doesn't send emails. To make it functional:

1. Set up a server to handle form submissions
2. Update the form action in `index.html`
3. Modify the form submission code in `script.js`
